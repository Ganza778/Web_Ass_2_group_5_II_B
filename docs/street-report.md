# Street Report - TourStack

## User Quote
> "I have a nice room in my home near Volcanoes National Park, but tourists always book expensive lodges in town. They don't even know we exist. I could offer them a real Rwandan experience for half the price." — Marie, home-stay owner in Musanze

## The Problem
Affordable home-stays in Musanze have no visibility in the digital space. Tourists rely on expensive international booking platforms that list only lodges and hotels, missing out on authentic cultural experiences and affordable accommodation options. Local hosts lose potential income, and tourists overpay for less authentic experiences.

## Chosen Scenario
**TourStack — Home-Stay Finder (Scenario B)**

A platform connecting tourists with local home-stay owners in the Musanze region.

## Scope for This Assignment
- User authentication with 3 roles (Host, Guest, Admin)
- Hosts can create, read, update, and delete home-stay listings
- Guests can browse and filter stays by price and cultural tags
- Booking request workflow with status tracking
- Live price calculator for bookings
- Host dashboard with statistics
- Admin verification system for new listings
- Responsive design for mobile and desktop
