# User Stories - TourStack

## Guest
1. As a guest, I want to browse all available home-stays so I can find accommodation.
2. As a guest, I want to filter stays by price range so I can stay within my budget.
3. As a guest, I want to filter by cultural tags (Imigongo/volcanic stone/traditional) so I can have an authentic experience.
4. As a guest, I want to see a live price estimate when booking so I know the total cost.
5. As a guest, I want to view my booking history so I can track my reservations.
6. As a guest, I want to cancel my booking if my plans change.

## Host
7. As a host, I want to create new listings so tourists can find my home-stay.
8. As a host, I want to edit my listings so I can update prices and availability.
9. As a host, I want to see booking requests so I can confirm or decline them.
10. As a host, I want to view statistics (total bookings, pending requests, revenue) so I can track my business.
11. As a host, I want my listings to be verified by admin so guests trust my stay.

## Admin
12. As an admin, I want to verify new listings so only legitimate stays are shown.
13. As an admin, I want to view all users so I can manage the platform.
14. As an admin, I want to reject inappropriate listings to maintain quality.
15. As an admin, I want to see all bookings to monitor platform activity.
